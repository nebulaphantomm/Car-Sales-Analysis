DriveCore: Advanced Automotive Sales Intelligence Dashboard
📌 Project Overview

DriveCore is a high performance Power BI solution designed to track global automotive sales, inventory health, and market distribution. This project moves beyond static data visualization by implementing a multi-theme user interface and interactive navigation systems. By analyzing a complex dataset of vehicle sales, the dashboard provides a 360 degree view of brand performance, engine trends, and regional sales density.

🛠️ Tech Stack
Business Intelligence Tool: Microsoft Power BI
Data Engineering: Power Query (Advanced ETL, data normalization, and schema optimization)
Analytical Engine: DAX (Dynamic measures for Inventory Value, Sales Trends, and Status Distribution)
UI/UX Design: High-fidelity Dark Mode interface with dynamic page navigation and conditional theme-switching
🚀 Advanced UX Features

Unlike standard reports, this dashboard implements professional software design principles:

Dynamic Page Navigation: Users can toggle between Blue, Gold, White, and Red themes to categorize different vehicle segments or performance tiers.
Interactive Overlay Menus: Implemented a hidden sidebar menu using Bookmarks and Selection panes, allowing for a "pop-up" navigation experience that maximizes screen real estate.
Contextual Slicers: Advanced filtering that updates global metrics across the entire "DriveCore" ecosystem.
🔍 Visual Walkthrough & Design Logic
1. Executive Performance Cards
Visuals: KPI Multi-row cards for Total Units Sold (1,550), Inventory Value (24.40T), and Vehicle Inventory (4,976).
Logic: Placed on the far left to establish immediate context. These high-level metrics serve as the "North Star" for executive decision-making.
2. Global Sales Distribution (Geospatial Analysis)
Visual: Bubble Map
Logic: Used to visualize market density across North America, Europe, and Asia. This helps stakeholders identify high-growth regions vs. underpenetrated markets.
3. Brand & Engine Performance
Visuals: Horizontal Bar Chart (Sales by Brand) and Cluster Bar Chart (Sales by Engine)
Logic: Comparison charts used to identify Nissan and Mercedes as lead performers, while the engine-year analysis tracks the transition in consumer demand over time.
4. Vehicle Status Distribution
Visual: Donut Chart
Logic: Provides an instant breakdown of inventory health—categorizing vehicles as Available, Reserved, or Sold to monitor supply chain efficiency.

💡 Business Impact
Inventory Optimization: Real-time visibility into "Available vs. Reserved" units prevents overstocking and capital lock-up.
Brand Strategy: Identifies top-tier brands (Nissan/Mercedes) to guide procurement and marketing spend.
Regional Growth: Pinpoints specific global hotspots where sales volume is peaking, allowing for localized promotional strategies.
Enhanced Engagement: The multi-theme UI and pop-up navigation increase user adoption by making complex data easy to navigate.
📥 How to Run
Clone this repository.
Open the .pbix file.
Interact: Use the color coded buttons at the top to switch between themes and click the Info icon to explore the pop-up menu system.
